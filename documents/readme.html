Readme

To compile the project the following has to be done:

  • Unzip project file
  • Open "ClientMandelBrot.java" and "ClientEuclideanTsp.java" in the clients package and hardcode the
    ip adress of the server into the constructors.
  • Open a commandline tool on both the server and client computer and move to the unzipped ClusterHW1
    folder
  • Run the "ant" command from the commandline
  • Run "ant runComputer" from the commandline on the server computer
  • Run "ant runTspClient" or "ant runMandlebrotClient" on the client computer

The result should show up on the screen.
